Import from Filesystem

You can easily import images directly from your local filesystem into the application.

Follow these steps to import images:

  1. Navigate to the File menu in the top navigation bar.
  2. Select "Import Images" from the dropdown options.
  3. Choose the files you wish to upload from the dialog box.
  4. Click Open to import the selected images.

Drag and Drop Import

For a faster workflow, you can drag and drop images directly into the application window.

To use this feature:

  1. Select one or more image files from your computer's file explorer.
  2. Drag the selected files over the application gallery.
  3. Drop the files to instantly start the upload process.

Viewing an Image

Double-clicking on any thumbnail in the gallery will open the image in a detailed view.

In the image view:

  1. You can see the full-resolution version of the image.
  2. Use the navigation arrows to move between images.

Viewing ExifTool Data

For advanced users, ExifTool data provides comprehensive information about the image file.

To access ExifTool data:

  1. Ensure the ExifTool panel is enabled in the view settings.
  2. Select one or more images in the gallery.
  3. Scroll through the detailed EXIF tags displayed in the panel.

Viewing Metadata

Inspect the technical details and selected metadata associated with your images.

To view metadata:

  1. Ensure the Metadata panel is enabled in the view settings.
  2. Select one or more images in the gallery.
  3. Review fields such as filename, rating and creation date.

Deleting Images

Remove unwanted images from your collection to keep it organized.

To delete an image:

  1. Select the image(s) you want to remove.
  2. Click the "Delete" button in the Edit menu or toolbar.
  3. Confirm the deletion in the popup dialog.
