Adding a Tag to a Filter

Filter your images by specific tags to find exactly what you're looking for.

To add a tag to a filter:

  1. Open the Filter topbar panel.
  2. Select a tag from the dropdown list or type to search.
  3. Click Add to add the tag at the end of the filter expression

Dragging Tags to Filter

You can intuitively build filters by dragging tags from the sidebar.

To use drag-and-drop filtering:

  1. Locate a tag in the Tags sidebar.
  2. Click and drag the tag to the filter area at the top.
  3. Drop it to apply the filter immediately.
Back to top ↑

Using Filter Operators

Combine multiple filters using logical operators like AND, OR, and NOT.

To use operators:

  1. Add multiple tags to the filter bar.
  2. Drag operator blocks between the tags to define the logic.
  3. Arrange them to create complex queries.

Saving Filters

Save your complex filter combinations for quick access later.

To save a filter:

  1. Construct your desired filter in the filter bar.
  2. Give your filter a name.
  3. Click the "Save" icon on the right side of the filter bar and confirm.
Back to top ↑

Loading Saved Filters

Quickly apply a previously saved filter configuration.

To load a saved filter you can choose three different methods:

  1. Double-click on the desired filter in the side panel.
  2. Drag the desired filter from the side panel to the filter expression.
  3. Select the desired filter in the filter dropdown.
Back to top ↑

Sharing Filters

Share your current filter configuration with others via a link.

To share a filter:

  1. Set up the filter you want to share.
  2. Click the "Share" button in the filter toolbar.
  3. A unique link is generated and copied in the clipboard.
Back to top ↑

Copying Filter Link

Copy the generated link to your clipboard to share it easily.

To copy the link:

  1. Find your shared filter in the sidebar.
  2. Click the "Share" button next to the filter name to copy the URL in the clipboard.
Back to top ↑

Viewing Shared Filters

Anonymous and registered users can see the collection generated with a shared filter.

To see a shared filter:

  1. Copy the URL in the clipboard as explained before.
  2. Paste it in the browser.
  3. You will only see the images matching the filter criteria.
Back to top ↑

Clearing Filters

Reset the view to show all images by clearing the current filters.

To clear filters:

  1. Click on "Clear" in the filter action buttons.
