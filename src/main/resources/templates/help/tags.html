Viewing All Tags

You can see all the tags currently used in your collection.

To view all tags:

  1. Open the Tags sidebar panel.
  2. Unselect all images in gallery.
  3. The tags side-panel shows all the tags in your collection.

Adding Tags

Tags allow you to categorize and easily retrieve your images later.

To add a tag:

  1. Select one or more images in the gallery.
  2. Type the tag name in the tags input field in the sidebar.
  3. Press Enter to apply the tag to the selected images.

Deleting Tags

If a tag is no longer needed or incorrect, you can remove it from an image.

To delete a tag:

  1. Select the image with the tag you want to remove.
  2. Select the tag in the tags list for that image.
  3. Click the "Delete" button in the Edit menu or toolbar.
  4. Confirm the deletion in the popup dialog.
